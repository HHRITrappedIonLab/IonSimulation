00物理系統概觀

模擬的對象是一群帶單位正電的離子，被關在線形 Paul trap裡，並用紅失諧的 369 nm 雷射從六個方向照射。三股作用同時進行：

作用物理在程式中的角色
阱的束縛RF 贗位能 + DC 端蓋 → 三維非等向諧振位能−(Kx·x, Ky·y, Kz·z)，Kx≪Ky=Kz
離子互斥庫倫力 ∝ 1/r²軟化後的成對加速度累加
雷射冷卻Doppler 都普勒冷卻 + 反衝加熱平均光壓力 + 隨機反衝擴散

結果：高溫亂跑的離子被雷射阻尼下來，最後在阱中央排成一條線形庫倫晶體（離子鏈）。這正是離子阱量子電腦裡 qubit 暫存器的樣子。

01溫度＝動能

「溫度」在這裡就是離子的平均動能。每個自由度按能量均分定理帶 \(\tfrac12 k_B T\)：

(1) \[ \langle E_k\rangle = \tfrac12 m\langle v^2\rangle = \frac{f}{2}\,k_B T \quad\Rightarrow\quad T \propto \langle v^2\rangle = v_{\mathrm{rms}}^2 \]

所以「讓離子變冷」＝「讓 \(v_{\mathrm{rms}}\) 變小」。畫面上的顏色（藍冷↔紅熱）、溫度計、以及專業版的 T (mK) 都直接由 \(v_{\mathrm{rms}}\) 換算。

02輻射壓與光子動量

每顆被吸收的光子帶動量 \(\hbar k\)（\(k = 2\pi/\lambda\)）。吸收後離子得到一記沿光束方向的動量 kick；隨後自發輻射把光子往隨機方向吐出。多次循環下：

  • 吸收的反衝總是沿著光束方向 → 有方向性 → 可造成淨力。
  • 自發輻射方向隨機 → 平均為零，但其漲落造成加熱（反衝擴散）。

單顆光子對速度的改變量 \(v_{\text{recoil}} = \hbar k/m\)。冷卻就是「方向性吸收」勝過「隨機反衝加熱」的結果。

03都普勒效應與選擇性散射

關鍵在於離子會動。在離子的參考系裡，看到的雷射頻率被都普勒平移；對應的有效失諧為：

(2) \[ \delta' = \delta - \mathbf{k}\cdot\mathbf{v} \quad,\quad \delta = \omega_L - \omega_0 \]

其中 \(\delta\) 是雷射對原子共振的失諧（紅失諧 \(\delta<0\)）。散射率採勞侖茲線型（Lorentzian）：

(3) \[ R(v) = \frac{\Gamma}{2}\cdot\frac{s}{1 + s + (2\delta'/\Gamma)^2} \]

\(s = I/I_{\text{sat}}\) 是飽和參數，\(\Gamma\) 是上能態的自然線寬。重點：散射率在 \(\delta' = 0\)（離子恰好「看到」共振）時最大。

選擇性的精髓：把雷射調成紅失諧（\(\delta<0\)）。當離子迎著光束跑（\(\mathbf{k}\cdot\mathbf{v}<0\)），\(\delta' = \delta - \mathbf{k}\cdot\mathbf{v}\) 被往 0 推 → 更容易散射；而順著光束跑的離子 \(\delta'\) 更偏離共振 → 較少散射。於是「迎面而來的離子」被優先打、被推回 → 減速。

04冷卻力與光學黏團（optical molasses）

沿一軸放一對對向光束，淨力是兩束散射力之差（力大小 \(= \hbar k\cdot R\)）：

(4) \[ F = \hbar k\,\bigl[\,R(\delta - kv) - R(\delta + kv)\,\bigr] \]

對小速度展開（一階）得到黏滯阻尼力：

(5) \[ F \approx -\beta v \quad,\quad \beta = -4\hbar k^2 s\,\frac{(2\delta/\Gamma)}{\bigl[\,1 + s + (2\delta/\Gamma)^2\,\bigr]^2} \]

對紅失諧 \(\delta<0\)：\(\beta>0\) → \(F = -\beta v\) 是阻尼 → 冷卻；對藍失諧 \(\delta>0\)：\(\beta<0\) → 反阻尼 → 加熱（模擬裡真的會看到離子越跑越快）。從六個方向都放光束，就能同時阻尼三個維度的運動 → 三維「光學黏團」。

共振（\(\delta=0\)）為何冷不下來？ 此時 \(\beta=0\)，淨阻尼消失；散射依然很強 → 只剩反衝加熱主導，溫度衝到高穩態。模擬裡把失諧滑桿放到 0 就能重現這個現象。

05都普勒冷卻極限

冷卻（阻尼）與加熱（反衝擴散 \(D\)）在穩態平衡：\(\langle v^2\rangle = D/2\beta\)。把兩者的顯式代入，最低溫出現在 \(\delta = -\Gamma/2\)、低飽和極限：

(6) \[ k_B T_D = \frac{\hbar\Gamma}{2} \quad\Rightarrow\quad T_D = \frac{\hbar\Gamma}{2k_B} \]

對 ¹⁷¹Yb⁺ 的 ²P1/2（壽命 ≈ 8.1 ns，Γ/2π ≈ 19.6 MHz）：

\[ T_D = \frac{\hbar\,(2\pi\cdot 19.6\,\text{MHz})}{2k_B} \approx \mathbf{0.47}\ \textbf{mK} \]

離子永遠有殘餘抖動，無法到絕對零度。模擬刻意保留這個地板（見 §11），專業版的溫度讀數也鎖在 \(T_D\) 以上。

06線形 Paul trap：贗位能與非等向束縛

真實線形阱用射頻（RF）四極場提供徑向束縛——交流電場本身不能靜態囚禁帶電粒子（Earnshaw 定理），但快速振盪的 RF 場時間平均後產生有效的贗位能（pseudopotential），等效於徑向諧振阱；軸向則靠DC 端蓋提供較弱的束縛。

離子在比 RF 慢的時間尺度上做secular（長期）運動，等效於三維非等向諧振子：

(7) \( \mathbf{a} = -(\,\omega_x^2 x,\ \omega_y^2 y,\ \omega_z^2 z\,) \) ，且  \( \omega_y,\omega_z \)（徑向，強）\( \gg \omega_x \)（軸向，弱）

因為軸向（x）束縛最弱，離子沿 x 軸排開成一直線。

兩種阱模型（可切換）：預設用 secular 贗位能近似（Kx, Ky, Kz 靜態諧振力），快又乾淨；專業版另提供「真實 RF」模式，顯式積分時變四極場 → 會出現真實的 micromotion（見下）。把 Ky=Kz 調到接近 Kx，會看到從線形鏈過渡到 zigzag、再到 3D 團狀晶體。

真實 RF 模式：顯式積分與 micromotion

RF 模式不再用時間平均，而是逐時積分時變四極場。標準型是 Mathieu 方程（\( \tau = \Omega t/2 \)）：

(7b) \[ \frac{d^2 u}{d\tau^2} + \big[\,a_u - 2q_u\cos(2\tau)\,\big]\,u = 0 \]

程式中實際積分的三維加速度（軸向 x 為 DC 靜態、徑向 y,z 為 DC 散焦＋反相 RF）：

(7c) \[ \ddot{x} = -K_x x,\quad \ddot{y} = \Big(\tfrac{K_x}{2} - A\cos\Omega t\Big)y + F_{\text{stray}},\quad \ddot{z} = \Big(\tfrac{K_x}{2} + A\cos\Omega t\Big)z \]

旋鈕 → secular 頻率（程式直接這樣算）：使用者設定物理旋鈕 — RF 頻率 \( \Omega \)、RF 電壓（\( A \propto V_{\text{RF}} \)）、DC 端蓋電壓（\( K_x \propto U_{\text{DC}} \)）；secular 頻率與 Mathieu \( (a,q) \) 都是算出來的結果：

(7d) \[ \omega_r = \sqrt{\frac{A^2}{2\Omega^2} - \frac{K_x}{2}},\quad \omega_z = \sqrt{K_x},\quad q = \frac{2A}{\Omega^2},\quad a = \frac{4K_x}{\Omega^2} \]

其中 \( A^2/2\Omega^2 \) 是 RF 贗位能給的徑向剛性，\( K_x/2 \) 是 DC 端蓋的徑向散焦。穩定需 \( q \lesssim 0.908 \) 且 \( \omega_r^2>0 \)，否則徑向失去束縛、離子流失（介面會即時標示）。把預設 \( \Omega,A \) 調回時，\( \omega_r \) 恰好回到 secular 模式的值，兩模式冷晶體一致。

micromotion：離子在 RF 頻率 \( \Omega \) 上的受迫快振，振幅正比於離 RF 零點（軸線）的距離：

(7e) \[ x_{\text{mm}} \approx \frac{A}{\Omega^2}\,\rho \approx \frac{q}{2}\,\rho, \qquad \rho = \sqrt{y^2+z^2} \]

所以：冷卻到軸線上（\( \rho\to0 \)）→ micromotion 消失；一個雜散 DC 場 \( F_{\text{stray}} \) 把平衡位置推到 \( \rho_{\text{eq}} = F_{\text{stray}}/\omega_r^2 \) → 即使冷晶體也有殘餘的 excess micromotion。這正是實驗室要「補償微運動」的情境（過量微運動造成加熱與二階都普勒頻移）。

單位錨定：介面把 \( \Omega \) 等角頻率乘上一個固定係數顯示成 MHz（錨定成預設 \( \Omega/2\pi\approx20 \) MHz），讓數字有實感（\( \omega_r/2\pi\sim\!3 \) MHz、\( \omega_z/2\pi\sim\!0.5 \) MHz）；絕對尺度是選定的，但比值與無因次的 \( (a,q) \) 是精確的。
實作要點：RF 振盪比 secular 運動快很多，需更小的時間步（RF 模式自動把 substeps 由 3 提到 9）。模擬刻意讓 \( \Omega \) 只比 secular 頻率高約一個數量級（非真實的 MHz），使 micromotion 看得見又能即時計算。溫度讀數是總動能（含 micromotion）——所以加大雜散場時溫度會上升，這在物理上正確（excess micromotion = 額外能量）。

07庫倫離子鏈

離子同性相斥（庫倫力），夠冷時動能不足以亂跑，於是停在「阱的束縛」與「彼此斥力」平衡的位置，形成規則的庫倫晶體。一維鏈的平衡位置由下式決定（無因次化後）：

(8) \[ u_m - \sum_{n<m}\frac{1}{(u_m-u_n)^2} + \sum_{n>m}\frac{1}{(u_m-u_n)^2} = 0 \]

解的特徵是中央間距較密、兩端較疏（中央離子被兩側更多鄰居往內擠）。模擬冷卻後量到的鏈正是這個分佈——這也是單元測試檢查的項目之一。

08運動方程與積分

每顆離子的總加速度是三項之和：

(9) \[ m\,\mathbf{a}_i = -\mathbf{K}\circ\mathbf{r}_i + \sum_{j\neq i}\frac{q^2}{4\pi\varepsilon_0}\,\frac{(\mathbf{r}_i-\mathbf{r}_j)}{|\mathbf{r}_i-\mathbf{r}_j|^3} + \mathbf{F}_{\text{laser}}(\mathbf{v}_i) \]

用半隱式（symplectic）Euler積分，每幀切成數個 substep 以保穩定：先用力更新速度，再用新速度更新位置。

physics.js · step()
// 每個 step 切成 sub 個子步，h = dt/sub
for (let st = 0; st < sub; st++) {
  // (1) 保守力 → 半隱式 Euler 更新速度
  for (let i = 0; i < ions.length; i++) {
    conservativeAccel(state, i, p, acc);     // 阱 + 3D 庫倫
    ions[i].vx += acc.ax * h; ions[i].vy += acc.ay * h; ions[i].vz += acc.az * h;
  }
  // (2) 雷射冷卻：平均光壓 + 反衝擴散（見 §9）
  // (3) 更新位置 + 速度/位置上限（數值保險）
  for (...) { ion.x += ion.vx * h; ... ; /* clamp 速度、球形邊界反彈 */ }
}

半隱式 Euler 對含阻尼的諧振系統穩定且便宜；substep + 速度上限 + 邊界反彈讓極端參數（超高光強、大量離子）也不會發散。

09雷射冷卻的實作：平均光壓 + 反衝擴散

真實過程是「離散的單光子散射事件」。但若直接對每次散射打一記 \(\hbar k\)，在遊戲尺度下這些離散 kick 的散粒雜訊會遠大於微弱的阻尼，導致冷卻不穩。因此本模擬把它拆成兩個在物理上對應的部分：

物理

① 平均輻射壓（系綜平均的散射力）
\(F_b = \hbar k\,n\cdot R(\delta')\)

② 反衝擴散（自發輻射的漲落）
動量在 3D 做隨機漫步，擴散係數 \(D \propto R\)

實作

① 對每束光累加確定性力
a += forceScale · L(δ′) · n

② 每步加一記隨機 3D 反衝
Δv = noiseAmp·√(Rtot·h)·N(0,1)·d̂

其中 \(L(\delta') = s/\bigl(1+s+(2\delta'/\Gamma)^2\bigr)\) 正比於散射率 \(R\)。對向光束的 \(L\) 之差自然給出 \(-\beta v\) 阻尼（§4）；隨機項給出 §5 的都普勒極限地板。兩者各自可調，於是能同時做到「冷卻夠快」又「地板乾淨可控」。

physics.js · step() 的雷射段
for (let b = 0; b < beams.length; b++) {
  const n = beams[b];
  const ndotv = n.nx*ion.vx + n.ny*ion.vy + n.nz*ion.vz;
  const dEff  = p.detuning - p.dopplerCoef * ndotv;   // δ′ = δ − k·v
  const x = 2 * dEff / gamma;
  const L = p.intensity / (1 + p.intensity + x*x);     // ∝ 散射率（勞侖茲）
  ax += p.forceScale * L * n.nx; ay += ...; az += ...;    // 平均光壓
  Rtot += L;
}
ion.vx += ax*h; ion.vy += ay*h; ion.vz += az*h;
if (Rtot > 0) {                                          // 反衝擴散（自發輻射）
  const sigma = p.noiseAmp * Math.sqrt(Rtot * h);
  const g = boxMullerGaussian(), d = randDir(rng);    // 3D 隨機方向
  ion.vx += sigma*g*d.x; ion.vy += sigma*g*d.y; ion.vz += sigma*g*d.z;
}
為什麼這是合理的近似：確定性項就是單光子散射力的系綜平均（期望值）；隨機項以正確的 √(R·dt) 標度重現反衝漲落。捨去的是單次散射的高頻散粒雜訊，換來數值穩定與可調的極限溫度——對教學/視覺化是划算且忠實的取捨。

10庫倫力與數值穩定性

成對庫倫加速度按 \(1/r^2\) 累加。為避免 \(r\to 0\) 的奇異與爆衝，做軟化（softening）與上限（cap）：

(10) \[ \mathbf{a}_i \mathrel{+{=}} C\,\frac{(\mathbf{r}_i-\mathbf{r}_j)}{\bigl(\,|\mathbf{r}_i-\mathbf{r}_j|^2 + \varepsilon^2\,\bigr)^{3/2}} \quad,\quad |\mathbf{a}| \le a_{\max} \]
physics.js · conservativeAccel()
const r2 = dx*dx + dy*dy + dz*dz + soft2;   // + ε² 軟化
const r  = Math.sqrt(r2);
let f = p.coulomb / r2;                      // ~1/r²
if (f > p.coulombFmax) f = p.coulombFmax;     // 上限保險
ax += f * (dx / r); ay += f * (dy / r); az += f * (dz / r);

軟化長度 \(\varepsilon\)（softening）也設定了晶體的最小間距尺度；coulomb 與阱剛性 Kx 的比值決定鏈長。\(N\le 14\) 時 \(O(N^2)\) 成對計算成本可忽略。

11單位換算：遊戲單位 → 物理量

模擬內部用任意「遊戲單位」（像素、幀）。專業版透過兩個錨點把它對應到 ¹⁷¹Yb⁺ 369 nm 的真實數字：

顯示量換算依據
失諧 δδ[MHz] = δ[Γ] × 19.6Γ/2π = 19.6 MHz（²P1/2 線寬）
溫度 TT = T_D · (vrms/vfloor)²T ∝ ⟨v²⟩，並以 T_D=0.47 mK 為地板錨定
飽和參數s = I/I_sat（即強度滑桿值）直接是模擬的 s
index.html · 溫度讀數（鎖在都普勒極限以上）
const GAMMA_MHZ = 19.6, T_D_MK = 0.47, RMS_FLOOR = 8;
// 速度地板對應 T_D；不可能低於極限：
const Tmk = T_D_MK * Math.pow(Math.max(rms, RMS_FLOOR) / RMS_FLOOR, 2);
誠實聲明：這是教學性的錨定，不是第一性原理的 SI 模擬。失諧的 MHz、溫度的 mK 量級正確且自洽，但絕對數值（如鏈長的 µm、阱頻的 MHz）是為了「好看、好玩、可恢復」而調的，並非特定實驗參數。

123D 投影與深度排序（無外部函式庫）

渲染完全手刻在 2D canvas 上，不依賴 three.js，維持單檔離線。世界座標先經攝影機旋轉（繞 y 軸 yaw、再繞 x 軸 pitch），再做透視投影：

(11) \[ \text{scale} = \frac{f}{D + z_{\text{cam}}} \quad,\quad (s_x, s_y) = (C + X\cdot\text{scale},\ C + Y\cdot\text{scale}) \]
index.html · project()
function project(x, y, z) {
  const cy=Math.cos(camYaw), sy=Math.sin(camYaw);
  let X = x*cy + z*sy, Z = -x*sy + z*cy, Y = y;       // 繞 Y (yaw)
  const cp=Math.cos(camPitch), sp=Math.sin(camPitch);
  const Y2 = Y*cp - Z*sp, Z2 = Y*sp + Z*cp;            // 繞 X (pitch)
  const depth = CAM_DIST + Z2, scale = FOCAL / depth;  // 透視
  return { sx: C + X*scale, sy: C + Y2*scale, scale, depth };
}

遮擋用畫家演算法（painter's algorithm）：把所有基元（離子球、電極棒段、端蓋環）收進一個陣列，依 depth 由遠到近排序後依序畫，近的覆蓋遠的。離子的大小、亮度也隨 scale 變化（近大近亮、遠小遠暗＝霧化）。

const prims = [];   // {kind:'rod'|'cap'|'ion', depth, ...}
// ...收集所有基元...
prims.sort((a,b) => b.depth - a.depth);            // 遠 → 近
for (const pr of prims) drawPrim(pr);
電極棒為何切成小段？一根與軸平行的長棒，旋轉後深度沿長度變化（一端近、一端遠）。把每根棒切成 ~18 小段、各自帶自己的 depth 一起排序，才能讓離子鏈正確地「擋住遠端、被近端擋住」，得到對的遮擋關係。

13電極與雷射的繪製

線形 Paul trap 電極

四根電極棒平行於軸（x），位於 (y,z) 平面上 45°/135°/225°/315° 的四極位置；對角一對標為 RF（銅金色），另一對為 接地（鋼灰色）；兩端各一個 DC 端蓋環。棒以漸層描邊 + 高光呈現金屬感，亮度隨深度霧化。

369 nm 雷射

專業版的光束固定畫成 369 nm 藍紫色（rgba(168,116,255,…)），用加色混合（'lighter'）畫成穿過阱的發光光柱——因為真實雷射不會因為 ~10 MHz 的失諧而改變可見顏色。童趣版才用紅↔藍變色來教「紅失諧冷卻 / 藍失諧加熱」。

離子

球體用徑向漸層（高光偏左上）+ 外圈光暈，顏色由速度映射（HSL：冷=青、熱=紅）。專業版加一點清晰高光（像離子螢光光點）、移除卡通臉；童趣版畫表情、雪花、尾巴。

14離子流失與載入：沒抓住就散掉 · 噴嘴 loading

有限阱深與穩定區：為什麼會「散掉」

離子阱能抓住離子是有極限的。在 secular 贗位能近似下徑向是一個諧振井，井深 \( D_r \sim \tfrac12 m\,\omega_r^2 R_\text{trap}^2 \)（\( R_\text{trap} \) ≈ 電極孔徑）。當離子徑向振幅超過 \( R_\text{trap} \)（撞上／越過電極），或阱根本不再束縛時，它就離開捕獲區、再也回不來——這就是離子流失（ion loss）。模擬有三條流失通道：

  • Mathieu 不穩定：當 \( q = 2A/\Omega^2 \gtrsim 0.908 \)，徑向本徵頻率變虛 → 振幅指數成長 → 離子被甩出。
  • 徑向不束縛：當贗位能剛性 \( K_\text{rad} = A^2/2\Omega^2 - K_x/2 \le 0 \)（RF 太弱或 DC 散焦太強），徑向沒有回復力。
  • 過熱／雜散場：把離子加熱到振幅超過 \( R_\text{trap} \)，或強雜散 DC 場把平衡點推進電極。

step() 每步用 secularFreqs 判定阱是否不穩定；不穩定時對每顆離子加一個向外反束縛加速度 \( \propto +K_\text{eject}\,\rho \)（重現虛頻率的指數發散；RF 模式本身也會自然發散）。一旦離軸距離 \( \rho=\sqrt{y^2+z^2} \) 超過逃逸半徑 escapeRho，該離子標記為 lost → 改為彈道飛行（不再受阱力／冷卻／庫倫力）→ 飛出 cullRadius 後移除並累計流失數。預設仍抓得住：正常冷卻時離子離軸 \( \lesssim 62 \) px < 逃逸半徑（\( \approx 78 \) px），所以一顆都不掉；但只要把 RF 電壓調高、RF 頻率調低（兩者都把 \( q \) 推過 0.908）或雜散場調大（把離子推離 RF 零點超過逃逸半徑），離子就會被甩出流失。

unstable = !secularFreqs(p).stable          // q≥0.908 或 Krad≤0
if (unstable) { ay += ejectK*y; az += ejectK*z }     // 反束縛 → 指數甩出
if (y*y + z*z > escapeRho*escapeRho) ion.lost = true  // 越過電極 → 流失

噴嘴 loading：原子爐 + 399 nm 光游離

真實離子阱用原子爐（oven）噴出一束中性 Yb 原子橫越阱中心，再用 399 nm 光游離雷射（¹S₀→¹P₁ 第一階）在中心把原子游離成 Yb⁺；只有在捕獲區內被游離、且夠慢能被冷卻接住的才留下，其餘穿過後被抽走。模擬忠實重現：

  • 每顆粒子帶 kind: 'neutral' | 'ion'。中性原子不受阱力、不被 369 nm 冷卻（躍遷不對）、不參與庫倫力，只做彈道運動，穿過阱後（離開 worldRadius）消失。
  • spawnAtoms() 從噴嘴口噴出朝中心、帶熱發散的中性原子束。
  • 光游離：399 nm 開啟時，中心游離區（半徑 piRadius）內的中性原子，每步以機率 \( 1-e^{-\,r_\text{pi}\,dt} \) 轉成離子，之後就受阱力與冷卻。
  • 捕獲：游離後若離子夠慢（徑向振幅 \( v_\perp/\omega_r < \) 逃逸半徑）就被接住、冷卻進鏈；太快或在邊緣被游離的就靠上面的機制散掉。399 nm 關 → 原子完全穿過、抓不到。
流失與載入是同一件事的兩面：loading 成功 = 冷卻有沒有在離子逃逸前接住它。把「離子數量」歸 0（空阱）後，可純用噴嘴從零抓起第一顆——正是真實實驗 load 離子的過程。
